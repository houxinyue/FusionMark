# Intel Arc A770 Windows 完整跑大模型指南（2026最新版）

**适用显卡**：Intel Arc A770 16GB（单卡或双卡）  
**推荐方式**：IPEX-LLM 官方 Ollama Portable Zip（无需安装 Python / oneAPI，10分钟上手）  
**支持模型**：7B~32B 量化模型（Q4/Q5）速度 20~40 tokens/s；双卡可跑 671B DeepSeek-R1 等  
**仓库状态**：官方仓库已于 2026年1月28日归档（只读），但下载链接和 Portable Zip 仍可正常使用。

---

## 1. 安装最新显卡驱动（必须第一步）

1. 打开浏览器访问 Intel 官网驱动页面：
   [https://www.intel.com/content/www/us/en/download/785597/intel-arc-graphics-windows.html](https://www.intel.com/content/www/us/en/download/785597/intel-arc-graphics-windows.html)
2. 下载最新版本（当前推荐 `gfx_win_101.8531.exe` 或更高）。
3. **以管理员身份运行**安装程序 → 全部下一步 → 重启电脑。
4. 重启后打开 **Arc Control** 或任务管理器确认 A770 已识别且驱动版本最新。

---

## 2. 下载并解压 Ollama Portable Zip

1. 访问官方 Releases 页面（v2.3.0-nightly）：
   [https://github.com/ipex-llm/ipex-llm/releases/tag/v2.3.0-nightly](https://github.com/ipex-llm/ipex-llm/releases/tag/v2.3.0-nightly)
2. 下载 **Windows 版 Portable Zip**（文件名类似 `ollama-portable-xxx.zip`）。
3. 解压到任意文件夹，例如：
   
   ```
   D:\ollama-intel
   ```

---

## 3. 配置环境变量（强烈建议）

打开 **命令提示符（cmd）**，进入解压文件夹：

```cmd
cd /d D:\ollama-intel
```

**逐行执行以下命令**（每次启动前都要设置）：

```cmd
set OLLAMA_NUM_GPU=999
set ONEAPI_DEVICE_SELECTOR=level_zero:0          :: 单卡用 0，双卡改成 level_zero:0;level_zero:1
set SYCL_PI_LEVEL_ZERO_USE_IMMEDIATE_COMMANDLISTS=1
set OLLAMA_NUM_PARALLEL=1                        :: 节省显存，推荐单任务
set OLLAMA_NUM_CTX=16384                         :: 上下文长度，可改 32768
set OLLAMA_MODEL_SOURCE=modelscope               :: 国内加速下载（推荐）
```

---

## 4. 启动 Ollama 服务

在同一个 cmd 窗口输入：

```cmd
start-ollama.bat
```

- 会弹出一个新窗口（服务窗口，不要关闭）。
- 第一次启动会编译内核，**可能卡 3~10 分钟**，请耐心等待。

---

## 5. 运行模型（新开一个 cmd）

**新开一个命令提示符**（不要关闭服务窗口），进入同一文件夹：

```cmd
cd /d D:\ollama-intel
ollama run deepseek-r1:7b
```

推荐入门模型（直接复制运行）：

- `ollama run qwen2.5:7b`（中文最强）
- `ollama run deepseek-r1:7b`
- `ollama run llama3.2:3b`（超轻量）
- `ollama run phi4`（微软最新）

下载完后直接聊天即可。

## 6. 推荐 GUI 界面（浏览器使用）

1. 下载 Open WebUI：
   - 去 https://github.com/open-webui/open-webui/releases 下载最新 `open-webui.exe`（Windows 便携版）
2. 双击运行 → 浏览器打开 http://localhost:8080
3. 选择你刚才跑的模型，即可获得 ChatGPT 式界面。

---

## 7. 常用进阶命令与调优

```cmd
:: 切换国内源更快下载
set OLLAMA_MODEL_SOURCE=modelscope

:: 双 A770 拼卡（32GB 显存）
set ONEAPI_DEVICE_SELECTOR=level_zero:0;level_zero:1

:: 省显存运行 32B+ 模型（把专家层扔到 CPU）
set OLLAMA_SET_OT="exps=CPU"

:: 超大上下文（推荐 32B 模型）
set OLLAMA_NUM_CTX=32768
:: 修改模型存储位置
set OLLAMA_MODELS=D:\ollama-intel\models
```

---

## 8. 监控与故障排除

- **GPU 监控**：打开 **Arc Control** → 查看显存占用和利用率。
- **第一次编译卡住**：正常现象，等 5~10 分钟即可。
- **显存不够**：用 Q4_K_M 量化，或加上 `OLLAMA_NUM_PARALLEL=1`。
- **服务启动失败**：以**管理员身份**运行 cmd，重装驱动，重启电脑。
- **模型下载慢**：必须加 `set OLLAMA_MODEL_SOURCE=modelscope`。
